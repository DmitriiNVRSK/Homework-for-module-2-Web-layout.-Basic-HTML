Introduction to frontend development

Frontend development development is responsible for creating the user interface of the user part of a website or web application. It includes the work of HTML, CSS and JavaScript.

Frontend development languages

  • HTML (Hyper Text Markup Language)-used to structure the content of a web page;
  • CSS (Cascading Style Sheets)- used to style a web page and create its appearance;
  • JavaScript- used to add interactivity to a web page such as animations, event handling, and dynamic content updates.

Frontend development tools

  1. Visual Studio Code- powerful code editor with support for many languages and plugins
  2. React- library for creating user interfaces
  3. Sass - CSS reprocessor with advanced features
  4. Webpack - module builder for JavaScript applications
